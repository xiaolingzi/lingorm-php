## 介绍及使用文档  
LingORM是为php编写的ORM框架，支持插入修改删除等基本ORM功能，也支持一般复杂的查询（比如复杂条件下的联表查询）。除此，也支持原生sql查询以应对更加复杂的情况。  
详细使用文档：[https://www.xxling.com/blog/article/3107.aspx](https://www.xxling.com/blog/article/3107.aspx)
